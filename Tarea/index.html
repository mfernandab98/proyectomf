Instalacion de Node Js

Comando para actualizar la terminal

sudo yum update

Instalacion del administrador de versiones nvm escribiendo el siguiente comando:

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.34.0/install.sh | bash

Se utiliza nvm para instalar Node.js , nvm puede instalar multiples versiones de Node.js y permitirle cambiar entre ellas. Activar nvm con el siguiente comando:

. ~/.nvm/nvm.sh

Se utiliza nvm para instalar la ultima version de Node.js con la siguiente linea de comando:

nvm install node

Probar que el Node.js esta instalado y funcionado correctamente escribiendo lo siguiente en la linea de comado:

node -e "console.log('Running Node.js ' + process.version)"

El siguiente comado es para repositorio

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.34.0/install.sh | bash

Instalacion de Github

Comando para instalar Github

sudo yum install git

Clonacion del codigo fuente

Clonar codigo fuente

git clone https://github.com/abkunal/Chat-App-using-Socket.io.git

Ubicarse en la carpeta del chat

cd Chat-App-using-Socket.io

Instalar librerias de Nodejs

npm install

Instalar pm2

npm install -g pm2
Configuracion de los servicios del Chat

Ejecutar pm2

pm2 start app.js

Probar ejecucion del codigo

curl 127.0.0.1:5000
